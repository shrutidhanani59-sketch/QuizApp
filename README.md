# 🎯 Quiz Application

This is Quiz Application build using HTML , CSS , and Javascript. next, Previous , submit button also add this application.

## 🚀 Features

📚 Multiple-choice questions

⏱️ Countdown timer for each question

✅ One answer selection per question

➡️ Next/Submit button navigation

📊 Final score display
